# Phase 6A: 前端功能补全与数据可看化设计

## 日期

2026-07-10

## 背景

前端副屏框架已有（Phase 4B/4C），但缺少 Agent 建议面板、LLM 复盘展示、种子数据。
打开前端大多显示"暂无数据"。

## 设计目标

1. 新增 Agent 实时建议面板（5s 轮询）
2. 新增 LLM 自然语言复盘展示
3. 种子脚本确保前端打开就有数据
4. 弹幕/告警端点 DB 无数据时 fallback 到模拟数据

## 新增 API

- GET /api/agent/suggestion — 播中 Agent 决策建议
- GET /api/review/llm/{room_id} — LLM 复盘总结

## 布局

顶部栏（房间/信任分/连接状态）
├── 讲解手卡（手动刷新）
├── Agent 实时建议（5s 轮询）
├── 弹幕洞察（10s 轮询）
├── 实时告警（10s 轮询）
└── 场次复盘 + LLM 总结（30s 轮询）
